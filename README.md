# Smart-Trash-Arduino
Caixote do lixo inteligente com Arduino UNO, sensor ultrassónico HC-SR04, servomotor e botão de fecho manual. A tampa abre automaticamente quando deteta a mão a menos de 10 cm, mantém-se aberta durante 12 segundos e pode ser fechada manualmente através de um botão de reset.
